## question-paper-website

## Tech Stack
Frontend: HTML5, CSS3

Backend: Python, Flask

Templating: Jinja2

## Folder Structure

/project-root
    /static
        styles.css
    /templates
        index.html
        semester.html
        subject.html
    app.py
    README.md

## How to Run

1. pip install Flask(if flask isn't previously installed)
2. python app.py
3. open browser and go to http://127.0.0.1:5000
